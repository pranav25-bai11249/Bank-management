The problem of this project addresses lack of simple,self-contained simulation environment for a banking system that allows a user to easily use banking system for creating a account and other tasks
Its scope is user interface with GUI, data security by verifying with account number and CIF number
Its target users are the people who simply wants to use banking software without any difficulty
Its high level features are creating a new account by name and contact number, allowing user to specify number of emi he/she want to pay, automatically close emi if all the emi are paid
